Supported languages

Computer Words supports any languages that the installed version of Pygments supports.

Extras: file names

As you may have noticed above, Computer Words extends the syntax of traditional Markdown code blocks by allowing attributes after the language name. Specifically, it supports one attribute, filename, which adds a line at the top of the code block.

Note: the attributes are parsed using a short, hacky function, not a proper parser, so the attribute's value must be an unquoted string with no spaces. This might be improved in the future but for now it pretty much works.

Extras: graphviz rendering

If you set the language to graphviz-dot-convert, and you have Graphviz installed, then Computer Words will render the contents to an SVG and insert the result. This site's own How it works page uses it to show diagrams.
